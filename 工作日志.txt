GSJuice后台管理系统
1.系统架构
前端:AdminLTE(Bootstrap)
后端:ThinkPHP5.1
数据库:MySQL、Redis(同步作用)
主要开发环境：Window(LAMP),Linux(LAMP或LNMP)
2.数据库设计(gs_ordering)
gs_classify|商品分类表
    id            分类ID
    store_id      店铺ID
    name          分类名称
    soft          排行
    lock          是否锁定
    add_time      添加时间
    update_time   更新时间

gs_config|系统配置表
    id            配置ID
    store_id      店铺ID
    top_image     顶部图片
    bottom_image  底部图片

gs_order|订单表
    order_id      订单ID
    store_id      店铺ID
    user_id       用户ID
    price         价格
    status        状态
    type          订单类型
    contact       联系人
    phone         联系方式

gs_order_detail|订单明细表
    order_id      订单ID
    product_id    商品ID
    quantity      数量
    amount        价格
    remark        备注

gs_product|商品
    product_id    商品ID
    name          商品名称
    price         商品价格
    description   商品描述
    image         商品图片
    tag           商品标签
    store_id      店铺ID

gs_store|店铺表
    id            店铺ID
    name          店铺名称
    address       店铺地址
    password      管理密码
    contact       联系方式
    start_time    起始时间
    end_time      结束时间

gs_user|用户表
    user_id       用户ID(openid)
    realname      真实姓名
    nickname      昵称
    sex           性别
    headImg       头像
    phone         手机号码
    birthday      出生日期

3.后台管理功能
(1).商品管理(添加、删除、修改)
(2).商品类别管理(添加、删除、修改)
(3).用户管理(查看)
(4).订单管理(添加、删除、修改)
(5).店铺数据分析
(6).店铺管理(管理员)
(7).系统配置
